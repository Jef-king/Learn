What is HTML?

* HTML stands for Hyper Text Markup Language.

* HTML is the standard markup language for creating Web pages.

* HTML describes the structure of a Web page.

* HTML consists of series of elements.

* HTML elements tell the browser how to display the content.

* HTML elements label pieces of contenet such as "this is a heading", "this is a paragraph", "this is a link" etc.

* HTML is not case sensitive.


<!DOCTYPE html>
// declaration defines that this document is an HTML5 document.
<html>
// element is the root element of an HTML page.
<head>
// elemnt contains meta information about the HTML page.
<title>Page Title</title>
// element specifies a title for the HTML page.
</head>
<body>
// element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks tables, list, etc.
<h1>This is the Heading</h1>
// element defines a large heading.
<p>This is the paragraph</p>
// element defines a paragraph.
</body>
</html>

HTML Documents

*

<!DOCTYPE html>

//Doctype declaration for html 5.

* All HTML documents must start with a document type declaration.

* The <!DOCTYPE> declaration represents the document type, and helps browsers to display web pages correctly.

* It must only appear once, at the top of the page (before any HTML tags).

* <!doctype> declaration is not case sensitive.

*

HTML Elements

* The HTML element is everything from the start tag to the end tag.

* <tagname>...............</tagname>

*

HTML Headings <h1> - <h6>

* <h1> defines the most important heading. <h6> defines least important heading.

* <h1>................</h1>

*

HTML Paragraphs <p>

* HTML paragraphs are defined with the <p> tag.

* <p>.................</p>

*

HTML Links <a>

* HTML links are defined with the <a> tag.

* The link's destination is specified in the href attribute.

* <a>..................</a>

HTML Images <img>

* HTML images are defined with the <img> tag.

* The source file (src), alternative text (alt), width, and height are provided as attributes

* <img src='' alt='' width='' height=''>

*

HTML Horizontal Rules <hr>

* The <hr> tag defines a thematic break in an HTML page, and is most often displayed as a horizontal rule.

* 

HTML Line Breaks <br>

* Use <br> if you want a line break (a new line) without starting a new paragraph.

* 

HTML Preformatted Text <pre>

* The text inside a <pre> element is displayed in a fixed-width font, and it preserves both spaces and line breaks.

* <pre>..........</pre>

*

HTML Style Attribute

* Setting the style of an HTML element, can be done with the style attribute.

* <tagname style='property:value;'>....</tagname>

*

HTML Formatting Elements

* <b> - Bold text. The HTML <b> element defines bold text, without any extra importance.

* <strong> - Important text. The HTML <strong> element defines text with strong importance.
  The content inside is typically displayed bold.

* <i> - Italic text. The HTML <i> element defines a part of text in an alternate voice or mood.
  The content inside is typically displayed in italic.

* <em> - Emphasized text. The HTML <em> defines emphasized text. The content inside is typically displayed in italic.

* <mark> - Marked text. The HTML <mark> elemnt defines text that should be marked or highlighted.

* <small> - Smaller text. The HTML <small> element defines smaller text.

* <del> - Deleted text. The HTML <del> element defines text that has been deleted from a document.
  Browser usually strike a line through deleted text.

* <ins> - Inserted text. The HTML <ins> elemnts defines a text that has been inserted into a document.
  Browsers will usually underline inserted text.

* <sub> - Subscript text. The HTML <sub> element defines subscript text. Subscript text appears half a character 
  below the normal line, and is sometimes rendered in a smaller font.

* <sup> - Superscript text. The HTML <sup> element defines superscript text. Superscript text appears half a character above the normal line, and sometimes rendered in a small font.

*

HTML Quotation and Citation Elements

* <blockquote> for Qutotations. The HTML <blockquote> element defines a section that is quoted from another source.
  Browser usually indent <blockquote> elements.
  <blockquote cite='google.com'>Hi</blockquote>

* <q> for Short Quotations. The HTML <q> tag defines a short quotation.

* <abbr> for Abbreviations. The HTML <abbr> tag defines an abbreviation or an acronym, like "HTML", "CSS".
  Marking abbreviations can give useful information to browsers, translation systems and search-engines.
  Egg <abbr title='World Health Organization'>WHO</abbr>

* <address> for Contact Information. The HTML <address> tag defines the contact information for the author/owner of a document or an article. The contact information can be an email address, URL, physical address, phone number, social media handle, etc. The text in the <address> element usually renders in italic, and browsers will always add a line break before and after the <address> element.

* <cite> for Work Title. The HTML <cite> tag defines the title of a creative work (e.g. a book, a song, a movie, a painting, a scultpture, etc.). The text in the <cite> element usually renders in italic.

* <bdo> for Bi-Directional Override. The HTML <bdo> tag is used to override the current text direction.
  <bdo dir='rtl'>This text will be written from right to left</bdo>

* 

HTML Comments <!--....-->

* Comments can be used to hide content.

* Comments are also great for debugging HTML, because you can commment out HTML lines of code, one at a time,
  to search for errors.

* Comments can be used to hide parts in the middle of the HTML code.

HTML Colors

* background-color you can set the background color for HTML elements.

* color you can set the color of text for HTML elements.

* border you can set the color of borders for HTML elements.
